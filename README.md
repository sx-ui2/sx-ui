# sx-ui

This repository publishes public release binaries and the install script for `sx-ui`.

The full source code is not published in this repository.

## Install

```bash
bash <(curl -Ls https://raw.githubusercontent.com/sx-ui2/sx-ui/main/install.sh)
```

## Releases

Download versioned packages from the [Releases](https://github.com/sx-ui2/sx-ui/releases) page.
